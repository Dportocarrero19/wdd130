PC Parts

General information

These are the most important parts for building a PC system. Purchase these parts at a micro center or computer store. Make sure each part will fit inside the computer case, as there can be different case and motherboard sizes. Make sure the computer parts work with each other effectively, some parts work better with certain manufacturers (intel or AMD). Handy tools for installation are a screwdriver and an anti-static wristband for sensitive electronic parts. For installation, please go to the installation page.

CPU

CPU - A Computer processing unit used for performing calculations, instructions and running programs.

RAM

Ram - (random access memory) short term memory where data is stored that needs to be used quickly.

Motherboard

Motherboard - Circuit board that ties all of the computer parts together and allows them to function.

Case

Case- Shell which safeguards all important components of computer system

RAM

CPU Cooler- device that removes heat from CPU and outside your case

Storage Drive

Storage drive- stores permanent computer data could be in the form of a HDD or SSD

Power Supply

Power supply- Pull power from outlet and converts AC to DC voltage for system

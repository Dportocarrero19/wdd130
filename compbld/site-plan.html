Content

Home page

The home page will be used for showing users the components they will need to build their computer system. Each component will have a picture, alongside basic information on what the components are used for. The components will be a processor (CPU), the case, a CPU cooler, a motherboard, graphics card, storage, random access memory (RAM), and a power supply. There will also be a section that mentions handy tools for building the computer, such as a screwdriver or an anti-static wristband. There will be a navigation bar on the top with the other two pages listed for installation and a FAQ.

Images for the Home page

Installation

The installation page will be used to show how to install each component and build the computer system. It will go step by step, telling the audience where each component will go. The description will be open-ended as each computer component is not the same and will vary between systems. The audience will be reminded that each component manufacturer has different layouts and should look at their manuals when building their systems. There will also be some tips and tricks PC builders should know of.

Images for the Page 2

FAQs

The FAQ will include help with common pc building questions. There could be many problems that arise from PC building and some may occur from specific components so the FAQ will be very general. Common solutions to many pc building problems can be somewhat obvious in hindsight, but could be overlooked in the moment (like cables not plugged in correctly).
